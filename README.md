# Laboratorio 1: Configuración del Entorno de Desarrollo Web

Este proyecto corresponde a la actividad **Post-Contenido 1** de la unidad **Fundamentos de la Web** El objetivo principal es configurar un entorno de desarrollo profesional y construir una base técnica sólida para el despliegue de aplicaciones web.

## 📝 Descripción
En este laboratorio se realizó la instalación y configuración de herramientas esenciales como Visual Studio Code, Git y GitHub. Además, se construyó una página web básica utilizando HTML5, CSS3 y JavaScript para validar el funcionamiento del entorno y practicar la inspección de elementos mediante Chrome DevTools

## 🛠️ Herramientas y Tecnologías
* **Editor de Código:** Visual Studio Code.
* **Control de Versiones:** Git y GitHub
* **Lenguajes:** HTML5, CSS3 y JavaScript
* **Navegador:** Google Chrome (DevTools)

### Extensiones de VS Code instaladas:
* **Live Server:** Para recarga automática del navegador.
* **Prettier:** Formateador de código predeterminado.
* **GitLens:** Para supercargar las capacidades de Git en el editor.
* **ESLint:** Para el análisis estático de código JavaScript.

La organización de archivos sigue el estándar sugerido para aplicaciones web básicas:

```text

```
